📝 HTML Implementation

Step 1: Include CSS and JavaScript
<!-- Include CSS -->
<link rel="stylesheet" href="assets/css/phone-formatter.css">

<!-- Include JavaScript -->
<script src="assets/js/phone-formatter.js"></script>
Step 2: Create HTML Container
<!-- Basic Usage -->
<div data-phone-formatter data-default-country="PK"></div>

<!-- With Custom ID -->
<div id="customer-phone" data-phone-formatter data-default-country="US"></div>
Step 3: Initialize (Optional)
<!-- Auto-initializes on DOM ready -->
<script>
// Manual initialization (optional)
window.AutoWorldPhoneFormatter.init();
</script>

🌍 Supported Countries

Country Code Format Example Validation Rules
🇵🇰 Pakistan +92 300-1234567 10-11 digits, starts with 3
🇺🇸 United States +1 (555) 123-4567 10 digits, valid area code
🇬🇧 United Kingdom +44 07123 456 789 10-11 digits, starts with 07
🇨🇦 Canada +1 (416) 123-4567 10 digits, valid area code
🇦🇺 Australia +61 4 1234 5678 9 digits, starts with 4
🇮🇳 India +91 98765 43210 10 digits, starts with 6-9
🇸🇦 Saudi Arabia +966 50 1234567 9-10 digits, starts with 5
🇦🇪 UAE +971 50 1234567 9 digits, starts with 5

⚡ JavaScript API

Get Formatted Phone Number
const phoneData = window.AutoWorldPhoneFormatter.getFormattedNumber('customer-phone');

// Returns object with:
console.log(phoneData);
// {
// countryCode: "+92",
// country: "Pakistan",
// formatted: "+92 300-1234567",
// raw: "3001234567",
// international: "+923001234567"
// }
Set Country Programmatically
window.AutoWorldPhoneFormatter.setCountry('customer-phone', 'US');
Create Phone Formatter Manually
const container = document.getElementById('phone-container');
window.AutoWorldPhoneFormatter.createPhoneFormatter(container);
